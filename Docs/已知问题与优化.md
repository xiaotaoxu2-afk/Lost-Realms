1. 前言
本项目核心玩法已完成，实现完整游戏流程，但存在部分小bug，不影响核心体验。本文档记录已知问题、原因，并制定优化计划，体现持续改进意识。

2. 已知小bug（不影响核心游戏体验）

bug现象1：蝙蝠敌人快速转向时动画卡顿（1-2帧）。
产生原因分析：未用对象池，实例化/销毁导致GC；动画参数切换不平滑。
暂未修复说明：卡顿短暂，不影响操作，后续优化。

bug现象2：部分攻击音效未触发（概率5%）。
产生原因分析：音频加载延迟，未加失败降级。
暂未修复说明：体验轻微影响，后续修复加载顺序。

bug现象3：背景图片显示错误。
产生原因分析：可能是Addressables导致的依赖丢失。
暂未修复说明：无实际影响，后续优化。

3. 潜在问题（未实际复现）

潜在问题1：长时间游戏后内存泄漏。
风险分析：Addressables释放遗漏，内存累积。
预防方案：用Memory Profiler测试，完善释放逻辑。
潜在问题2：低配置PC帧率波动。
风险分析：未针对低端设备优化。
预防方案：加画质设置，低模式关闭特效。

4. 后续优化计划（分阶段实施）
第一阶段：修复小bug（1-2周）

优化内容：UI自适应。
具体实施方法：加Layout Group和Content Size Fitter，调整锚点。
预期效果：支持多种分辨率，无错位。
优化内容：音效稳定性。
具体实施方法：优化加载顺序，加失败降级。
预期效果：触发率99%以上。

第二阶段：性能优化（2-3周）

优化内容：内存泄漏修复。
具体实施方法：Profiler测试，完善释放逻辑。
预期效果：内存稳定，无累积。
优化内容：画质设置。
具体实施方法：加选项，低模式关闭特效/降低帧率。
预期效果：低配置60FPS稳定。
优化内容：战斗打击感。
具体实施方法：加屏幕震动、击退飘字。
预期效果：反馈清晰，提升体验。

第三阶段：功能扩展（3-4周）

扩展内容：新手引导。
具体实施方法：第一关加UI提示，引导核心操作。
预期效果：降低学习成本。
扩展内容：多语言支持。
具体实施方法：ScriptableObject存储文本，支持切换。
预期效果：适配海外玩家。
